Settings

Hugging Face token

Needed once for gated models (pyannote diarization). Create a free token at huggingface.co/settings/tokens, and accept the terms of pyannote/speaker-diarization-3.1 and pyannote/segmentation-3.0. The token is stored locally and only used to download models.

Compute device

Changing device takes effect the next time models are loaded.

Job history

Finished jobs (transcript + original audio) are kept on disk so playback and editing survive restarts. Oldest jobs beyond this limit are deleted automatically (3–20, default 5).
